## 1. 通用目标检测数据集

### 1.1 COCO (Common Objects in Context)

**配置文件**: `coco.yaml`

**基本信息**:
- **数据集大小**: 20.1 GB
- **训练图像**: 118,287 张
- **验证图像**: 5,000 张
- **测试图像**: 20,288 张
- **类别数量**: 80 类

**类别列表**:
包含常见物体类别，如：person, bicycle, car, motorcycle, airplane, bus, train, truck, boat, traffic light, fire hydrant, stop sign, parking meter, bench, bird, cat, dog, horse, sheep, cow, elephant, bear, zebra, giraffe, backpack, umbrella, handbag, tie, suitcase, frisbee, skis, snowboard, sports ball, kite, baseball bat, baseball glove, skateboard, surfboard, tennis racket, bottle, wine glass, cup, fork, knife, spoon, bowl, banana, apple, sandwich, orange, broccoli, carrot, hot dog, pizza, donut, cake, chair, couch, potted plant, bed, dining table, toilet, tv, laptop, mouse, remote, keyboard, cell phone, microwave, oven, toaster, sink, refrigerator, book, clock, vase, scissors, teddy bear, hair drier, toothbrush

**路径配置**:
```yaml
path: coco
train: train2017.txt
val: val2017.txt
test: test-dev2017.txt
```

**用途**:
- 最广泛使用的目标检测数据集
- 适合训练通用目标检测模型
- 涵盖日常生活中的常见物体

**使用建议**:
- 推荐作为基础模型的预训练数据集
- 适合迁移学习的起点
- 支持目标检测和实例分割任务

**示例用法**:
```bash
yolo train data=coco.yaml
```

**文档链接**: https://docs.ultralytics.com/datasets/detect/coco/

## 使用技巧

### 1. 快速开始
对于初学者，建议从小型数据集开始：
```bash
# 目标检测
yolo train data=coco8.yaml model=yolov8n.pt epochs=3

# 图像分割
yolo train data=coco8-seg.yaml model=yolov8n-seg.pt epochs=3

# 姿态估计
yolo train data=coco8-pose.yaml model=yolov8n-pose.pt epochs=3

# 旋转目标检测
yolo train model=yolov8n-obb.pt data=dota8.yaml epochs=3
```

### 2. 数据增强
所有数据集都支持 Ultralytics 的数据增强功能：
```yaml
# 在训练命令中添加增强参数
yolo train data=coco.yaml model=yolov8n.pt \
  hsv_h=0.015 hsv_s=0.7 hsv_v=0.4 \
  degrees=0.0 translate=0.1 scale=0.5 \
  shear=0.0 perspective=0.0 \
  flipud=0.0 fliplr=0.5 mosaic=1.0
```

### 3. 多GPU训练
对于大型数据集，使用多GPU训练：
```bash
yolo train data=coco.yaml model=yolov8n.pt device=0,1,2,3
```

### 4. 自定义数据集
基于现有配置创建自定义数据集：
```yaml
# custom.yaml
path: /path/to/your/dataset
train: images/train
val: images/val
test: images/test

names:
  0: class1
  1: class2
  2: class3
```

### 5. 数据集转换
使用 Ultralytics 工具转换其他格式：
```python
from ultralytics.data.converter import convert_coco

# 转换 COCO 格式到 YOLO 格式
convert_coco(labels_dir='path/to/coco/annotations')
```

## 常见问题

### Q1: 如何选择合适的数据集？
A: 根据您的应用场景、可用计算资源和数据集大小进行选择。初学者建议从 COCO128 或 COCO8 开始。

### Q2: 数据集下载失败怎么办？
A: 某些数据集（如 Argoverse, xView）需要手动下载。查看配置文件中的下载说明。

### Q3: 如何使用自己的数据集？
A: 参考现有 YAML 文件格式，创建自己的配置文件，确保图像和标注格式正确。

### Q4: 多光谱和灰度数据集有什么区别？
A:
- 灰度：单通道 (channels: 1)
- 多光谱：多通道 (channels: 10)
- 标准RGB：3通道 (默认)

### Q5: OBB 数据集与普通检测数据集的区别？
A: OBB (Oriented Bounding Box) 数据集支持旋转边界框，适合航拍图像等需要角度信息的场景。
